# Stuyvesant Freshman caucus 25-26 website

## Development

### Building

1. Install dependencies: `pnpm install`
2. Run build command `pnpm build`

- To run dev server, run `pnpm dev`

### Starting a new file

- Use `<main>`
- Use classes
  - flex
  - min-h-screen
  - w-full
  - px-4
  - py-8
  - sm:px-6
  - lg:px-8
